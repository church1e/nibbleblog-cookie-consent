# cookie_consent
Simple plugin script for nibbleblog to handle the EU Cookie Law consent.


# Install
1. Copy the "cookie_consent" folder into your Nibbleblog's plugin directory.
2. Install the "EU Cookie consent" plugin via the Nibbleblog Dashboard.
